# AutoTeam
TShock plugin that automatically joins players to a configurable team when they join the server.
## Configuration
For config value `Team`:
| value     | desc |
| :---      | :---
| **``1``** | ``Red`` |
| **``2``** | ``Green`` |
| **``3``** | ``Blue`` |
| **``4``** | ``Yellow`` |
| **``5``** | ``Pink`` |
